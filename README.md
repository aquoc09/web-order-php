# web-order-php

Dự án PHP sử dụng Composer để quản lý web order.

## Yêu cầu hệ thống
- PHP >= 8.1
- Composer
- MySQL 
- Wampserver64

## Cài đặt Composer Packages
Chạy lệnh:
composer install

Lệnh này sẽ tự động tải toàn bộ thư viện bên trong file `composer.json`.

## Development
- Chạy `composer update` để cập nhật thư viện
- Chạy `composer dump-autoload -o` để tối ưu autoload
